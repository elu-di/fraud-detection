# Scripts

This directory contains standalone Python scripts for reproducible pipeline execution.

| Script | Purpose |
|---|---|
| `run_preprocessing.py` | End-to-end data cleaning and feature engineering |
| `run_modeling.py` | Model training, evaluation, and saving artifacts |
| `run_explainability.py` | SHAP analysis and plot generation |

## Usage

```bash
# From the project root
python scripts/run_preprocessing.py
python scripts/run_modeling.py
python scripts/run_explainability.py
```
